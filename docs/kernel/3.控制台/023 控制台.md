# 转义字符

转义字符通常不能看见的字符, 但是被打印的时候会也许出现相应的效果.

在使用显卡的字符模式打印字符时, 直接写入像 `\n` (换行符), 不会有效果, 换行的效果是在操作系统中人为添加的.

像这样的字符, 我想要首先实现:

- `\n` -> 换行
- `\r` -> 清除本行此字符之前的字符, 并将光标移动到行首
- `\f` -> 将光标往下移动一行
- `\b` -> 清除此字符之前的一个字符, 仅对本行有效
