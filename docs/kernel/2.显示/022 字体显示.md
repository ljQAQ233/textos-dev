# Overview

## 点阵字库

顾名思义, 就是点阵形式的字库, 对于一个字符, 我们只需要保存它在每一个像素的状态就行了, 使用一位来表示一个像素, 1 就是 显示, 0 则反之
举个例子:

`I` be-like: (星号是1 , 点是0)

```plaintext
********
...*....
...*....
...*....
...*....
...*....
...*....
********
```
